An embeddable AI robot guide for any website.

RobotGuide adds a draggable, pose-reactive robot character to your site. It follows scroll, reacts to mouse movement with 24 distinct poses, types commentary as sections come into view, and optionally connects to your AI backend for live-generated speech.

Two files. Zero dependencies. Works with any framework or plain HTML.

Get Started View on GitHub

What it does

Everything you need. Nothing you don't.

Built as a single self-contained JS module with a companion CSS file.

🖱️ Mouse-reactive poses

24 named poses that respond to mouse speed, direction, and proximity. Debounced so it doesn't flicker.

📜 Scroll spy

Automatically moves to and comments on sections as they scroll into view. No setup beyond listing your section IDs.

✋ Draggable

Users can drag the robot anywhere on screen. It stays pinned until they navigate to a new section.

🤖 AI commentary

Point it at any POST endpoint and get live AI-generated speech. Falls back to your static lines if the endpoint is offline.

🎨 Fully themeable

Bring your own character images. All styles are scoped to .rg-active — no conflicts with your existing CSS.

📦 Zero dependencies

Vanilla JS. No React, no jQuery, no build step required. Drop in two files and call RobotGuide.init().

Quick start

Two files, one function call.

Include the CSS in your <head> and the JS before </body>.

<!-- 1. Styles -->
<link rel="stylesheet" href="robot-guide.css" />

<!-- 2. Script -->
<script src="robot-guide.js"></script>

<!-- 3. Init -->
<script>
const guide = RobotGuide.init({
  sections: ['hero', 'features', 'pricing', 'contact'],

  poses: {
    basePath: '/your-images/robot-poses/',
    // map defaults to the 24 named poses — override any subset
  },

  commentary: {
    hero:     ["Welcome! I'll be your guide today."],
    features: ["These features took a while to build. Worth it."],
    pricing:  ["No hidden fees. Scout's honor."],
    contact:  ["You made it this far — just say hi."],
  },

  // Optional: live AI commentary
  aiEndpoint: '/api/section-explain',

  draggable: true,
  scrollSpy: true,
});

// Public API
guide.say("Hello from the robot!");
guide.moveTo('pricing');
guide.setPose('happy-cheer');
</script>

Reference

Configuration & API

sections

Array of element IDs the robot reacts to. Order determines scroll-spy priority.

poses.basePath

Base URL prefix for pose images. Combine with the default map of 24 named poses.

commentary

Object of sectionId → string[]. Lines cycle. Falls back to "Interesting section.".

aiEndpoint

URL of a POST endpoint. Receives { section, context }, expects { text } in response.

guide.say(text)

Make the robot say anything programmatically with the typewriter effect.

guide.moveTo(id)

Programmatically move the robot to any section element by ID.

guide.setPose(name)

Force a specific pose by name. See the full pose list in the README.

guide.destroy()

Remove the robot from the page and clean up all listeners.
