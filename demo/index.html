An embeddable AI robot guide for any website.

RobotGuide adds a draggable, pose-reactive robot character to your site. It follows scroll, reacts to mouse movement with 24 distinct poses, types commentary as sections come into view, and optionally connects to your AI backend for live-generated speech.

One folder. Three HTML lines. Zero dependencies.

Get Started View on GitHub

What it does

Everything you need. Nothing you don't.

Built as a single self-contained JS module with a companion CSS file.

🖱️ Mouse-reactive poses

24 named poses that respond to mouse speed, direction, and proximity. Debounced so it doesn't flicker.

📜 Scroll spy

Automatically moves to and comments on sections as they scroll into view. No setup beyond listing your section IDs.

✋ Draggable

Users can drag the robot anywhere on screen. It stays pinned until they navigate to a new section.

🤖 AI commentary

Point it at any POST endpoint and get live AI-generated speech. Falls back to your static lines if the endpoint is offline.

🎨 Fully themeable

Bring your own character images. All styles are scoped to .rg-active — no conflicts with your existing CSS.

📦 Zero dependencies

Vanilla JS. No React, no jQuery, no build step required. Copy the folder, edit the config, done.

Quick start

One folder. One config file. Three HTML lines.

Copy the robot-guide/ folder into your site root, edit robot-guide.config.js, and add three lines to your HTML. The robot initialises itself — no extra script required. 

<!-- Step 1: copy robot-guide/ folder into your site root -->
<!-- Step 2: edit robot-guide/robot-guide.config.js         -->
<!-- Step 3: add these three lines (order matters):         -->

<link rel="stylesheet" href="robot-guide/robot-guide.css" />
<script src="robot-guide/robot-guide.config.js"></script>
<script src="robot-guide/robot-guide.js"></script>

<!-- That's it. The robot starts automatically. -->

Inside robot-guide.config.js you'll find:

window.RobotGuideConfig = {

  sections: ['hero', 'features', 'about', 'contact'],

  poses: {
    basePath: 'robot-guide/poses/',
  },

  commentary: {
    hero:    ['Your tagline. Very short.', 'Make it punchy.'],
    contact: ['Just say hi.', 'One form. No spam.'],
  },

  // aiEndpoint: '/api/section-explain',
  // Optional — POST { section, context } → { text }
  // Falls back to static commentary above when not set.

  draggable: true,
  scrollSpy: true,
  headerSelector: 'header',
  sectionSnapOffsetX: -78,

};

Need to control the robot from your own JS? The public API is always available after init:

window.robotGuide.trigger('contact');      // move + comment
window.robotGuide.say('Custom text!');    // say anything
window.robotGuide.setPose('happy-cheer'); // force a pose
window.robotGuide.destroy();              // remove from page

Reference

Configuration & API

sections

Array of element IDs the robot reacts to. Order determines scroll-spy priority.

poses.basePath

Base URL prefix for pose images. Combine with the default map of 24 named poses.

commentary

Object of sectionId → string[]. Lines cycle. Falls back to "Interesting section.".

aiEndpoint

URL of a POST endpoint. Receives { section, context }, expects { text } in response.

window.robotGuide.say(text)

Make the robot say anything programmatically with the typewriter effect.

window.robotGuide.trigger(id)

Move the robot to a section and trigger its commentary in one call.

window.robotGuide.setPose(name)

Force a specific pose by name. See the full pose list in the README.

window.robotGuide.destroy()

Remove the robot from the page and clean up all listeners.
